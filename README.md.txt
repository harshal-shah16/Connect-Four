
<h1 align="center">Connect Four Game</h1>

<br/>

<h2>About</h2>
Developed popular 2-player Connect 4 game in Python using Turtle graphics.
The first to get 4 tokens in a row vertically or diagonally wins !
There are two modes : Human vs Human or Human vs Computer
Players are designated as Red, Yellow or Computer
At the end of each game, winner is alloted additional one point and score incremennted 
and saved in a txt file



<h2>Installation</h2>

1. Download this project as zip and extract it
2. Run the file named connect_four_driver using Python IDLE
